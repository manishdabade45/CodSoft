# Codsoft Internship Tasks 🚀

Welcome to my repository for the **Codsoft Web Development Internship**. This repository contains three advanced, responsive, and aesthetically pleasing web projects developed as part of my internship curriculum.

---

## 📂 Project Structure

### 🏅 Task 1: Personal Portfolio
- **Description:** A premium, professional portfolio website designed for an AI & ML Engineer. It features smooth scrolling, intersection observers for reveal animations, and a modern dark-themed aesthetic.
- **Key Features:**
  - Modern Sticky Navigation.
  - Dynamic Skill Grid.
  - Project Showcase with Hover Effects.
  - Fully Responsive Design.
- **Location:** [./Task-1/](./Task-1/)

### 👟 Task 2: Adidas Landing Page Clone
- **Description:** A high-fidelity UI clone of the Adidas store interface. It implements core design principles of premium e-commerce brands, including high-impact hero sections and modular product grids.
- **Key Features:**
  - Announcement Bars.
  - Interactive Filter Chips.
  - Clean, Modular Product Cards.
  - Premium Typography and Color Palette.
- **Location:** [./Task-2/](./Task-2/)

### 🔢 Task 3: Interactive Calculator
- **Description:** A functional and minimalist web-based calculator. Designed with a sleek layout and intuitive user experience for basic arithmetic operations.
- **Key Features:**
  - Clean Geometric Layout.
  - Responsive Button Grid.
  - Real-time Computation Logic.
- **Location:** [./Task-3/](./Task-3/)
